# JP Technologies

**AI-powered products for cognitive enhancement, productivity, and research.**

A professional portfolio showcasing ventures in life coaching, agent platforms, research engines, and knowledge infrastructure.

## Products

### 🔮 BetterLife
iOS AI life coach powered by Claude. Structured cognitive behavioral therapy, habit tracking, and personalized coaching for neurodivergent individuals and high-performance teams.

- **URL**: https://betterlife.care/
- **Status**: Live
- **Tech**: SwiftUI, StoreKit 2, OpenRouter API

### ⚙️ AgentForge
Distributed agent orchestration platform. Multi-provider LLM routing, real-time streaming, MCP integration, and visual dashboard.

- **URL**: https://agentforge-beta-three.vercel.app/
- **Status**: Active Development
- **Tech**: Go, TypeScript, Supabase

### 🧬 Helix
AI-powered research engine combining SearXNG meta-search with Claude reasoning. Structured research with citations and subscription tiers.

- **URL**: https://vane.sytes.net/
- **Status**: Live
- **Tech**: Next.js 15, Supabase, Stripe

### 🍽️ BiteSaver
AI-powered recipe vault and meal planning system. Coming soon.

- **Status**: In Development
- **Target**: Q3 2026

### 🔧 OpenClaw
Multi-channel agent control system with Telegram, Discord, and web integrations.

- **Status**: Active
- **Tech**: Go, MCP, Composio

## Getting Started

### Local Development

```bash
# Start local server
npm run dev
# Opens at http://localhost:3000
```

### Build & Deploy

```bash
# Preview build
npm run preview

# Deploy to Vercel (git push auto-deploys)
git push origin main
```
